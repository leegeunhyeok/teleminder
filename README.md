## Installation

```bash
npm install -g telenoty
```

## Usage

- Required [Bot](#bot)
  - Create your Telegram bot
  - Get access token
  - Get chat id

```bash
# Regist bot information (save on ~/.telenoty)
# --regist, -r
telenoty --regist {BOT_TOKEN} {CHAT_ID}

# Send message
# --message, -m
telenoty --message {MESSAGE}
```
